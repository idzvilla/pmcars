## Обзор

Новая отдельная страница `/info/kalkulyator-rashod` — калькулятор полных расходов на импорт авто из США в Беларусь. Показывает разбивку по четырём блокам: аукцион, доставка, услуги компании, растаможка. Реактивный пересчёт без кнопки.

Существующий таможенный калькулятор (`/info/kalkulyator`) остаётся без изменений.

- EUR: `https://api.nbrb.by/exrates/rates/451`

## Результаты (правая колонка, sticky)

Заголовок: **"Итого всех затрат"** + суммарная цифра в USD.

### Блок 1: Расходы на аукционе

| Строка | Расчёт |
|--------|--------|
| Стоимость авто | Введённая цена в USD |
| Аукционный сбор | По таблице из `auction-fees.json` (тир по цене) + фиксированные сборы платформы |

### Блок 2: Расходы на доставке

| Строка | Расчёт |
|--------|--------|
| Доставка по США до порта | Из `us-domestic-prices.json` по (локация + аукцион + размер ТС) |
| Фрахт (океан) | Из `shipping-config.json → ocean` по ключу `ПОРТ_США-ПОРТ_НАЗНАЧЕНИЯ`, тариф RORO |
| Страховка | 1% от стоимости авто, минимум $50 |
| Портовый сбор | Клайпеда: $350 / Поти: $400 |
| Доставка до города | Из `shipping-config.json → inlandToBY`, тариф `runner` |

### Блок 3: Наши услуги

| Строка | Значение |
|--------|----------|
| Наши услуги | 1000 BYN (константа, легко менять) |

### Блок 4: Расходы на растаможку

| Строка | Расчёт |
|--------|--------|
| Таможенная пошлина | Из `calculateTotal()` (существующий `src/lib/calculator.ts`), результат в EUR |
| Таможенный сбор | 120 BYN (константа) |
| Утилизационный сбор | 1089 BYN (для авто старше 3 лет) / 544 BYN (до 3 лет) |
| СВХ | 650 BYN (константа) |

> Примечание: `calculateTotal()` принимает `engineVolume` в куб. см. Пользователь вводит литры — конвертация: `литры × 1000`.

### Итого

Все суммы конвертируются в USD через курсы НБРБ:
- BYN → USD: `сумма / usdRate`
- EUR → USD: `сумма × eurRate / usdRate`

Под итогом — дисклеймер: "Расчёт является ориентировочным. Точную стоимость уточняйте у менеджера."

## Combobox для площадок

Поскольку в CSV ~600+ локаций, обычный `<select>` неудобен. Компонент `Combobox`:
- Текстовый input с поиском (фильтрация по введённым символам)
- Dropdown со списком совпадений (макс. 50 в видимой области, скролл)
- При смене аукциона — список фильтруется, выбранная площадка сбрасывается
- Реализация через управляемый state без внешних библиотек

---

## Реактивность

Нет кнопки "Рассчитать". Результаты пересчитываются через `useEffect` при каждом изменении любого поля формы. Если обязательные поля не заполнены (цена = 0, площадка не выбрана, объём = 0) — результаты показывают нули.

---

## Валюты в результатах

Результаты отображаются в исходных валютах (как у westmotors):
- Стоимость авто, аукционный сбор, доставка, фрахт, страховка, портовый сбор — в **USD**
- Пошлина — в **EUR**
- Таможенный/утилизационный сбор, СВХ, наши услуги — в **BYN**
- Итого — в **USD** (всё конвертируется)

## Данные для конвертации CSV

Скрипт конвертации (запускается один раз, результат коммитится):
```
CSV колонки: [id, location, auction, port, regular, large, oversize]
→ JSON: [{id, location, auction, port, regular, large, oversize}]
```
Пустые значения (0,0,0) включаются — локация просто вернёт 0 для доставки.
