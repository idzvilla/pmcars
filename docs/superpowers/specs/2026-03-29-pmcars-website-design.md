# pmcars.by — Дизайн-документ сайта

**Дата:** 2026-03-29
**Проект:** Корпоративный сайт компании pmcars.by
**Описание:** Многостраничный сайт для компании по импорту автомобилей из США в Беларусь

---

## 1. Контекст и позиционирование

Компания pmcars.by занимается привозом автомобилей из США в Беларусь под ключ. Целевая аудитория — как частные клиенты (авто для семьи), так и перекупщики (авто для перепродажи).

**Ключевое конкурентное преимущество:** специализация исключительно на США — в отличие от конкурентов (westmotors, autogroup, solutionauto), которые работают с США + Китаем + Кореей + Европой одновременно. Это нишевая экспертиза, которую нужно подчёркивать.

**Референсы:**
- `references/solutionauto.by` — хедер (структура, фиксированный, двухрядный)
- `references/autogroup.by` — общий визуал, чистота, шрифты
- `references/westmotors` — структура главной, видеоотзывы, команда

## 9. Страница «Информация» — подстраницы

### Таможенный калькулятор (`/info/kalkulyator`)

Полностью фронтовый расчёт (без бэкенда), логика идентична autogroup.by.

**Поля ввода:**
- Стоимость авто (EUR)
- Объём двигателя (куб. см)
- Тип транспорта: Автомобиль / Мотоцикл
- Тип двигателя: Бензин/Дизель / Электро
- Возраст авто: До 3 лет / От 3 до 5 лет / Более 5 лет
- Чекбокс «Указ №140» (льгота — пошлина делится на 2)

**Фиксированные сборы (конвертируются в EUR по курсу НБРБ):**
- Таможенный сбор: 120 BYN
- Утилизационный сбор до 3 лет: 544 BYN
- Утилизационный сбор 3+ лет: 1089 BYN
- ЭПТС: 70 BYN
- Курс EUR берётся с API НБРБ: `https://api.nbrb.by/exrates/rates/451`

**Логика расчёта таможенной пошлины:**

*Авто до 3 лет* — берётся максимум из двух методов:
- По стоимости: ≤8500 EUR → 54%, >8500 EUR → 48%
- По объёму: ≤8500→2.5, ≤16700→3.5, ≤42300→5.5, ≤84500→7.5, ≤169000→15, >169000→20 (EUR/куб.см)

*Авто 3–5 лет* (только по объёму, EUR/куб.см):
≤1000→1.5, ≤1500→1.7, ≤1800→2.5, ≤2300→2.7, ≤3000→3.0, >3000→3.6

*Авто старше 5 лет* (только по объёму, EUR/куб.см):
≤1000→3.0, ≤1500→3.2, ≤1800→3.5, ≤2300→4.8, ≤3000→5.0, >3000→5.7

*Электромобиль:* пошлина = 0

*Мотоцикл:*
- объём ≤800: базовая = стоимость × 0.15; объём >800: базовая = стоимость × 0.10
- Итого = базовая + (стоимость + базовая) × 0.20

**Указ №140:** пошлина делится на 2

**Вывод результатов:**
- Стоимость авто
- Таможенная пошлина
- Таможенный сбор
- Утилизационный сбор
- ЭПТС
- **Итого**

### ЭПТС (`/info/epts`)
- Что такое ЭПТС
- Как получить
- Сроки и стоимость

### ДКП (`/info/dkp`)
- Что такое договор купли-продажи
- Что должно быть в ДКП при покупке авто из США
- Образец/шаблон

- Порядок действий
